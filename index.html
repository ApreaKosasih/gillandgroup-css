Breakpoints :

Sistem grid dibuat berdasarkan enam tingkatan grid default, kode dari breakpoint sebagai berikut :

  • Sangat Kecil (xs) Keterangan ukuran layar : <576px

  • Kecil (sm) Keterangan ukuran layar : >576px

  • Sedang (md) Keterangan ukuran layar : >768px

  • Lebar (lg) Keterangan ukuran layar : >992px

  • Sangat Lebar (xl) Keterangan ukuran layar : >1200px

Grid options :

Untuk penggunaan lebih spesifik seperti memastikan pada layar ponsel atau layar desktop, gunakan class col-breakpoints-value value dari columns grid berjumlah 12 dan kode dari breakpoint sebagai berikut :

  • Sangat Kecil (xs) Keterangan ukuran layar : <576px

  • Kecil (sm) Keterangan ukuran layar : >576px

  • Sedang (md) Keterangan ukuran layar : >768px

  • Lebar (lg) Keterangan ukuran layar : >992px

  • Sangat Lebar (xl) Keterangan ukuran layar : >1200px

Contoh penggunaan : col-sm-6 spesifik untuk layar kecil dengan ukuran 576px - 768px dengan jumlah column grid sebanyak 6

Penggunaan utilities :

Background color

Penggunaan background color bisa di gunakan di dalam satu baris class dengan card menggunakan class bg-warna contoh : class="card bg-secondary" sedangkan untuk warna dari title dan deskripsi menggunakan class text-warna contoh : class="card-title text-primary"

Extra :

Alignment

Penggunaan alignment menggunakan class ta-nilai keterangan: ta = text-align keterangan nilai : c = center | l = left | r = right | j = justify contoh : class="card-title ta-c"

Penggunaan utilities :

Border radius

Class button default memiliki border radius dengan ukuran 8px, untuk merubah style dari border menggunakan br-nilai contoh : class="btn-primary br-none" untuk merubah style button agar tidak memiliki border radius, dan juga bisa menggunakan nilai lain seperti : br-none | br-round | br-rounded

Color

Untuk memanggil class button harus diikuti dengan kode pemanggilan warna background seperti : btn-warna contoh : class="btn-primary" sedangkan untuk warna dari text default berwarna dark dan untuk merubah bisa menggunakan class text-warna contoh : class="btn-edu-navy text-light"

Extra :

Font Size

Untuk merubah ukuran text bisa menggunakan class font-ukuran keterangan: font = font-size keterangan ukuran : sm = 1rem * 0.75 | md = 1rem | lg = 1rem * 1.5 | xl = 1rem * 2 | xxl = 1rem * 3 contoh : class="btn-primary text-light font-md"

Penggunaan utilities :

Color

Class button outlined secara default memiliki style yang sama dengan button, yang membedakanya adalah pada background yang defaultnya adalah transparant dan warna justru diterapkan pada border, perbedaan paling menonjol pada button outlined adalah pada style ketika keadaan hover, jadi pada button outlined pemanggilan button harus diikuti dengan warna text yang sama dan memaggil class untuk text ketika di hover contoh : class="btn-outlined-primary text-primary text-hover-light"

Extra :

Font Size

Untuk merubah ukuran text bisa menggunakan class font-ukuran keterangan: font = font-size keterangan ukuran : sm = 1rem * 0.75 | md = 1rem | lg = 1rem * 1.5 | xl = 1rem * 2 | xxl = 1rem * 3 contoh : class="btn-primary text-light font-md"
